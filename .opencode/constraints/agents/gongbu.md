# Gongbu (公部/实现代理) 约束

## 完整实现

方法体必须完整实现，禁止以下情况：
- TODO 注释：// TODO: implement
- 占位符：throw new NotImplementedError()
- 不完整代码块：只写了函数签名
- 示意代码：// placeholder

每个方法都必须有实际的逻辑实现。

## 无省略分支

所有代码分支必须实现：
- if/else 的所有分支都要实现
- switch 语句需要 default 分支
- 异常情况必须有 try-catch 处理
- 边界情况必须有判断和处理

## 类型明确

避免使用 any 类型：
- 禁止使用 any（除非万不得已，需要注释说明原因）
- 函数参数和返回值都要标注类型
- 字段和变量的类型都要声明
- 泛型参数要使用具体类型

## 错误恢复

遇到运行时错误能自动恢复：
- 网络错误：有重试机制（3 次）
- 文件错误：检查文件存在性
- 解析错误：有降级方案
- 超时错误：有超时处理
